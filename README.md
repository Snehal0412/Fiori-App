# RAP Managed Fiori Application

## Overview
End-to-end SAP ABAP RAP Managed application with Fiori Elements UI.

## Architecture
- RAP Programming Model (Managed)
- CDS View Entities
- Behavior Definitions & Implementations
- OData V5 Service
- Fiori Elements UI

## Technologies
- ABAP RAP
- CDS (View Entities)
- OData V5
- SAP Fiori Elements
- Eclipse ADT

## Features
- Create / Update / Delete
- Draft handling
- Validations & Determinations
- Managed implementation

## How to Run
1. Activate CDS & Behavior objects
2. Publish Service Binding
3. Launch Fiori app from service binding
